## 6. Guidelines

### Do
- Use 4px grid for all spacing
- Use `#155DFC26` for primary actions and CTAs
- Stick to 3 font weights: 400, 600, 700
- Use `Source Sans Pro` as the primary typeface

### Don't
- Don't introduce colors outside the palette above
- Don't mix fonts beyond Source Sans Pro and IBM Plex Mono
- Don't use inline styles when the design system covers the pattern
- Don't use border-radius values outside: 0px 12px 12px 0px, 4px, 6px, 8px, 12px, 25.6px, 50%, 3.35544e+07px
