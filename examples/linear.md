# Design System: Linear – The system for product development

> Extracted from [https://linear.app](https://linear.app) by brandmd

## 1. Visual Theme & Atmosphere

**Overall mood:** Balanced and professional

**Density:** Moderate. The layout uses a varied spacing scale.

**Shape language:** Rounded, friendly aesthetic with generous corner radii.

**Depth:** Uses 5 shadow styles for layering and elevation.

## 2. Color Palette & Roles

- **Light gray** (`#E6E6E6`) — Surface / card background
- **Dark Muted Blue** (`#08090A`) — Dark background / footer
- **Light Green** (`#00FF0512`) — Surface / card background
- **Dark Blue** (`#5E6AD2`) — Accent background
- **Red** (`#EB5757`) — Accent background
- **Cyan** (`#06B6D4`) — Accent background
- **Light Muted Cyan** (`#F7F8F8`) — Light text (on dark)
- **Dark Muted Blue** (`#62666D`) — Secondary text
- **Muted Blue** (`#8A8F98`) — Secondary text
- **Pink** (`#F79CE0`) — Link / accent text
- **Dark Muted Blue** (`#24282C`) — Divider / border

## 3. Typography Rules

**Primary font:** Inter Variable
**Secondary font:** Berkeley Mono

**Size scale:**
- Headings: 24px, 32px, 48px, 64px
- Body / UI: 14px, 15px, 16px, 18px, 20px
- Captions / Small: 10px, 11px, 12px, 12.25px, 13px, 13.3333px

**Weights in use:** 300, 400, 510, 590

## 4. Component Stylings

### Buttons
- Corner radius: 2px
- Primary action color: `#5E6AD2` (Dark Blue)

### Cards
- Shadow: `rgba(0, 0, 0, 0.03) 0px 1.2px 0px 0px`
- Corner radius: 2px
- Background: `#E6E6E6`

### Inputs
- Border color: `#24282C`
- Corner radius: 2px

## 5. Layout Principles

**Spacing scale:** 2px, 4px, 5px, 6px, 8px, 12px, 24px, 32px

**Base unit:** 4px grid (values are mostly multiples of 4).

**Border radii:** 2px, 4px, 6px, 8px, 12px, 12px 12px 0px 0px, 22px, 50%
