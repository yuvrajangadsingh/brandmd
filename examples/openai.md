## 1. Visual Theme & Atmosphere

**Overall mood:** Balanced and professional

**Density:** Moderate. The layout uses a varied spacing scale.

**Shape language:** Rounded, friendly aesthetic with generous corner radii.

**Depth:** Uses 1 shadow style for layering and elevation.

## 2. Color Palette & Roles

- **White** (`#FFFFFF`) — Page background
- **Black** (`#0D0D0D14`) — Dark background / footer
- **Green** (`#04B84C26`) — Accent background
- **Black** (`#000000E6`) — Primary text
- **Dark gray** (`#5D5D5D`) — Secondary text
- **Dark gray** (`#8F8F8F`) — Secondary text
- **Black** (`#0D0D0D0D`) — Divider / border
- **Light gray** (`#EDEDED`) — Divider / border

## 3. Typography Rules

## 5. Layout Principles

**Spacing scale:** 2px, 4px, 6px, 8px, 12px, 16px, 20px, 24px

**Base unit:** 4px grid (values are mostly multiples of 4).

**Border radii:** 0px 0px 8px 8px, 6px, 8px, 8px 8px 0px 0px, 10px, 12px, 16px, 9999px

## 6. Guidelines

### Do
- Use 4px grid for all spacing
- Use `#04B84C26` for primary actions and CTAs
- Stick to 3 font weights: 400, 500, 600
- Use `OpenAI Sans` as the primary typeface

### Don't
- Don't introduce colors outside the palette above
- Don't mix fonts beyond OpenAI Sans and SF Mono
- Don't use inline styles when the design system covers the pattern
- Don't use border-radius values outside: 0px 0px 8px 8px, 6px, 8px, 8px 8px 0px 0px, 10px, 12px, 16px, 9999px
