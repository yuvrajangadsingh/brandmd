## 6. Guidelines

### Do
- Use 4px grid for all spacing
- Use `#FF173F2D` for primary actions and CTAs
- Stick to 4 font weights: 400, 500, 600, 700
- Use `aBCFavorit` as the primary typeface

### Don't
- Don't introduce colors outside the palette above
- Don't mix fonts beyond aBCFavorit and domaine
- Don't use inline styles when the design system covers the pattern
- Don't use border-radius values outside: 4px, 6px, 8px, 10px, 12px, 16px, 24px, 3.35544e+07px
