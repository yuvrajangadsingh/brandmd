## 6. Guidelines

### Do
- Use 4px grid for all spacing
- Use `#146EF5` for primary actions and CTAs
- Stick to 4 font weights: 400, 500, 550, 600
- Use `WF Visual Sans Variable` as the primary typeface

### Don't
- Don't introduce colors outside the palette above
- Don't mix fonts beyond WF Visual Sans Variable and WFVisualSans-Mono
- Don't use inline styles when the design system covers the pattern
- Don't use border-radius values outside: 0px 0px 8px 8px, 0px 2px 2px 0px, 0px 0px 4px 4px, 2px 2px 0px 0px, 2px, 4px, 8px, 50%
